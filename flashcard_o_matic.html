Project: Flashcard-O-Matic

Create & study virtual flashcards

"Flashcard-O-Matic" is a digital flashcard web app designed for educational use in schools, as a homework tool, or for personal use. Teachers are able to create decks of cards based on their subject of choice, and students can flip through the deck to study. The main focus of this project was practice with state management and rendering in React for CRUD operations (create, read, update, delete), APIs with hooks, React Router, and basic styling using Bootstrap.

This was a great opportunity in learning to "lift up" state through component props, create nested routes, run tests from the command line, and form control. Please visit the GitHub repo here to see screenshots of app features and learn more about my development process.
