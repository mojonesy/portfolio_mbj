Project: We Love Movies

Movie/theater database & server

"We Love Movies" is a server and database intended for users to gain access to data about movies, theaters, and reviews. The purpose of this project was practice in building complex routes with Node and Express, managing migrations with Knex as an SQL query builder, and creating and managing a PostgreSQL database.

I created middleware and error handlers, recieved requests and accessed information through routes and query parameters, built an API following RESTful design principles, wrote database queries to complete CRUD requests in an Express server, returned joined and nested data using Knex migrations, and deployed my server using Heroku.
