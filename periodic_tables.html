Project: Periodic Tables

Restaurant reservation management system

"Periodic Tables" is a restaurant reservation system intended to assist restaurant staff with booking, editing, seating, searching, and cancelling reservations and tables. My final capstone project with Thinkful, it is a full stack web app built over the course of 3 weeks. The process was a fulfilling and rewarding opportunity to use all the skills I learned over my program together to create one coherent and fully functioning application.

The client side was built with React.js, JavaScript, and styled with Bootstrap and CSS. I gleaned inspiration from Dribbble.com for the modular theme, did some basic color theory research to choose my color scheme, and chose a basic Adobe font for typography. The server was built using Node.js, Express.js for routing and middleware, and Knex.js/PostgreSQL for connecting to the database. I used ElephantSQL to create 4 database instances for development, testing, preview, and production environments.
